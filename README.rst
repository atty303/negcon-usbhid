
Overview
========

* This is an USB HID adapter for `neGcon`_ based on `V-USB`_.

.. _neGcon: http://en.wikipedia.org/wiki/NeGcon
.. _V-USB: http://www.obdev.at/products/vusb/index.html


Appendix
========

Negcon Cable Layout
-------------------

====== ======= ======
Color  Pin No. Signal
------ ------- ------
Brown  1       DAT
Orange 2       CMD
Black  4       GND
Red    5       VCC
Yellow 6       SEL
Blue   7       CLK
Green  9       (ACK)
====== ======= ======

====== ======= ======

Brown  7       CLK
